# Actividad 1 – Introducción a la Programación Web

En esta actividad se aplicaron los conceptos básicos de programación web mediante el uso de **HTML, CSS, JavaScript y XML**.

Se creó una página web que muestra la diferencia entre **HTML** como lenguaje de presentación y **XML** como lenguaje de almacenamiento de datos.  
Además, se diseñó una tarjeta de perfil con estilos modernos usando **CSS**, y se implementaron funciones interactivas como un **contador** y un **buscador** utilizando **JavaScript nativo**.

El proyecto fue desarrollado en **Visual Studio Code** y organizado siguiendo una estructura clara de carpetas.
