HTML vs XML

¿Qué es HTML?

HTML es un lenguaje que se utiliza para mostrar información en páginas web.

¿Qué es XML?

XML es un lenguaje que se utiliza para almacenar y organizar datos.

Diferencia

  • HTML muestra información
  • XML guarda datos
