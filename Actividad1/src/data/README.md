# Actividad 1 – Introducción a la Programación Web

Este proyecto corresponde a la práctica de la asignatura
Introducción a la Programación Web.

## Contenido del proyecto

- HTML: estructura de la página web
- CSS: diseño moderno usando Flexbox y variables
- JavaScript: contador interactivo y buscador de palabras
- XML: almacenamiento de datos de productos
- VS Code: entorno de desarrollo utilizado

## Estructura del proyecto

/Actividad1
/docs     -> Tutorial teórico en HTML
/src      -> Código fuente del proyecto
/css      -> Estilos CSS
/js       -> Lógica JavaScript
/data     -> Datos XML

El proyecto fue desarrollado y probado en Visual Studio Code.
